# Лабораторна робота 3 - Web застосунок

## Назва системи

MakiivskyRodnichokForever

## Короткий опис системи

MakiivskyRodnichokForever - це система контролю споживання води, яка поєднує серверну частину, базу даних, Web-клієнт, мобільний застосунок та IoT-пристрої. Система збирає показники лічильників води, зберігає історію споживання, розраховує суму до сплати, відображає сповіщення про критичні події та підтримує звернення користувачів до адміністратора.

## Прийняті інженерні рішення

Front-end частину системи реалізовано як React-застосунок з використанням Tailwind CSS. React обрано для побудови компонентного інтерфейсу, а Tailwind CSS - для швидкого створення адаптивного та підтримуваного дизайну без окремої великої CSS-структури.

Застосунок взаємодіє з backend через REST API. Після входу користувач отримує JWT-токен, який додається до захищених запитів у заголовку `Authorization: Bearer`. Роль користувача визначається відповіддю сервера. Якщо роль дорівнює `Admin`, відкривається адміністративний інтерфейс; для ролі `User` відкривається користувацька панель.

## Будова Web-застосунку

Web-застосунок складається з таких частин:

1. Екран входу та реєстрації.
2. Користувацький інтерфейс.
3. Адміністративний інтерфейс.
4. API-клієнт на базі `fetch`.
5. Модуль локалізації українською та англійською мовами.
6. Модуль експорту, імпорту та резервного копіювання.
7. Маршрутизатор `react-router-dom`, який розділяє застосунок на окремі сторінки.

Основні маршрути користувача: `/app/dashboard`, `/app/devices`, `/app/readings`, `/app/alerts`, `/app/tickets`, `/app/profile`.

Основні маршрути адміністратора: `/admin/dashboard`, `/admin/users`, `/admin/devices`, `/admin/tariffs`, `/admin/tickets`, `/admin/alerts`, `/admin/backups`, `/admin/logs`.

## Користувацький інтерфейс

Звичайний користувач може переглядати власні пристрої, історію показників, суму до сплати, сповіщення та звернення до підтримки. Також користувач може створити нове звернення і редагувати власний профіль.

## Інтерфейс адміністратора

Адміністратор може керувати користувачами, пристроями, тарифами, зверненнями, сповіщеннями та переглядати системні логи. Також реалізовано створення резервної JSON-копії даних, експорт локальних налаштувань і імпорт тарифів.

## Інтернаціоналізація та локалізація

У застосунку підтримуються українська та англійська мови. Дата й час форматуються через `Intl.DateTimeFormat` відповідно до вибраної локалі: `uk-UA` або `en-US`. Грошові значення форматуються через `Intl.NumberFormat` у валюті UAH.

Порядок сортування текстових значень реалізовано через `Intl.Collator`, тому імена, назви пристроїв і тарифи сортуються відповідно до обраної мови. Також підтримано напрями інтерфейсу та введення тексту: LTR, RTL і auto.

## Висновок

Розроблений Web-застосунок є повноцінною front-end частиною системи MakiivskyRodnichokForever. Він має окремі інтерфейси для користувача та адміністратора, взаємодіє з backend через REST API, підтримує локалізацію, регіональні формати, резервне копіювання, експорт та імпорт даних.
